# Prompt - Dịch truyện phương Tây (Anh → Việt)

## Thể loại

Fantasy, Sci-fi, Romance, Thriller, Light Novel phương Tây

## Quy tắc dịch

### Tên riêng

- **Giữ nguyên** tên tiếng Anh của nhân vật
  - VD: Harry Potter, Aragorn, Katniss
- **Địa danh nổi tiếng:** giữ nguyên
  - VD: London, Hogwarts, Middle-earth
- **Tên tự đặt trong truyện:** có thể Việt hóa nếu phù hợp
  - VD: Winterfell → Thành Winterfell

### Xưng hô

- **Hiện đại:** anh/em, cô/cậu, ông/bà
- **Cổ trang phương Tây:** ngài/bệ hạ, thưa ngài, thưa điện hạ
- **Gia đình:** ba/mẹ, anh/chị/em
- **Tước hiệu:** giữ nguyên + giải thích
  - VD: Duke → Công tước, Earl → Bá tước, Lord → Lãnh chúa

### Phong cách văn chương

**Fantasy:**

- Giữ không khí huyền ảo, kỳ ảo
- Thuật ngữ magic: nhất quán (mana, spell → ma lực, phép thuật)
- Tên spell: giữ nguyên + dịch nghĩa lần đầu
  - VD: "Expelliarmus" → "Expelliarmus - phép tước vũ khí"

**Sci-fi:**

- Thuật ngữ khoa học: dịch chính xác
- Tên công nghệ: giữ nguyên hoặc Việt hóa nhất quán
  - VD: hyperspace → không gian siêu tốc, warp → dịch chuyển

**Romance:**

- Tương tự ngôn tình nhưng giữ văn hóa phương Tây
- Không áp dụng xưng hô cổ trang Trung Quốc

### Thuật ngữ thường gặp

| English    | Dịch                | Ghi chú |
| ---------- | ------------------- | ------- |
| Magic      | Ma thuật/Phép thuật |         |

| Party      | Tổ đội              |         |

### Văn hóa phương Tây

- Giữ nguyên reference văn hóa phương Tây
- Thành ngữ: tìm equivalent Việt hoặc dịch nghĩa
  - VD: "break a leg" → "chúc may mắn"
  - VD: "piece of cake" → "dễ như ăn kẹo"
- Không áp dụng điển tích Trung Quốc

### Light Novel Nhật (bản English)

- Nếu biết gốc Nhật, ưu tiên âm Nhật
- Giữ phong cách light novel: hài hước, tự sự ngôi thứ nhất
- Thuật ngữ anime/manga: giữ nguyên cộng đồng hiểu

## Input format

```
Thể loại: [fantasy/sci-fi/romance/...]
Glossary (nếu có):
{bảng thuật ngữ}

Đoạn cần dịch:
{nội dung tiếng Anh}
```

## Output

Chỉ trả về bản dịch tiếng Việt hoàn chỉnh.

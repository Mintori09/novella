# Prompt - Dịch truyện Tiên hiệp / Huyền huyễn

## Thể loại

Tiên hiệp, Huyền huyễn, Tu tiên, Võ hiệp

## Quy tắc dịch

### Tên riêng

- Tên nhân vật: chuyển sang âm Hán Việt chuẩn
  - VD: 叶凡 → Diệp Phàm, 萧炎 → Tiêu Viêm
- Địa danh, môn phái: âm Hán Việt
  - VD: 青云门 → Thanh Vân Môn, 天斗帝国 → Thiên Đấu Đế Quốc
- Bảo vật, công pháp: âm Hán Việt + giải thích ngắn lần đầu
  - VD: 九转玄功 → Cửu Chuyển Huyền Công

### Cảnh giới tu luyện

Dùng thuật ngữ quen thuộc trong cộng đồng tiên hiệp Việt:

| Trung  | Hán Việt      |
| ------ | ------------- |
| 炼气期 | Luyện Khí Kỳ  |
| 筑基期 | Trúc Cơ Kỳ    |
| 金丹期 | Kim Đan Kỳ    |
| 元婴期 | Nguyên Anh Kỳ |
| 化神期 | Hóa Thần Kỳ   |
| 炼虚期 | Luyện Hư Kỳ   |
| 合体期 | Hợp Thể Kỳ    |
| 大乘期 | Đại Thừa Kỳ   |
| 渡劫期 | Độ Kiếp Kỳ    |

### Phong cách văn chương

- Câu văn mang không khí cổ trang: trang trọng, cổ kính
- Không dùng từ hiện đại, slang trong bối cảnh cổ trang
- Miêu tả cảnh vật: thơ mộng, hùng vĩ
- Miêu tả chiến đấu: kịch tính, nhịp nhanh
- Đối thoại giữa cao nhân: tôn kính, lễ nghĩa

### Thành ngữ & Điển tích

- Ưu tiên thành ngữ Hán Việt tương đương
- Điển tích: giữ nguyên + chú thích ngắn lần đầu
- Thuật ngữ đạo giáo, phật giáo: giữ âm Hán Việt

### Xưng hô

- Sư phụ - đệ tử: sư phụ/sư tôn - đồ nhi/đệ tử
- Đồng môn: sư huynh/sư đệ/sư tỷ/sư muội
- Tiền bối - vãn bối: tiền bối - vãn bối/hậu bối
- Vợ chồng: phu quân/phu nhân, lang quân/nương tử

## Input format

```
Glossary (nếu có):
{bảng thuật ngữ đã thiết lập}

Đoạn cần dịch:
{nội dung tiếng Trung}
```

## Output

Chỉ trả về bản dịch tiếng Việt hoàn chỉnh.
